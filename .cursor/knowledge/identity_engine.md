# Identity Engine

The identity engine models:
- Who the user is
- Who they want to become
- What roles they play
- What attributes define them
- How they evolve over time

## Core Elements

### Roles
Examples:
- Leader
- Father/Mother
- Warrior
- Creator
- Builder
- Strategist

### Attributes
Strengths, values, tendencies, behaviors.

### Aspirations
Long-term goals and character traits the user wants.

### Identity Actions
Actions that reinforce or contradict identity are tracked.

### Identity Score
A probabilistic model that:
- Shifts based on actions
- Evolves from personal journaling
- Influences XP multipliers
- Guides coaching responses

The engine is designed to feel like the user is **growing into the best version of themselves**, with Pulse OS as a partner.
