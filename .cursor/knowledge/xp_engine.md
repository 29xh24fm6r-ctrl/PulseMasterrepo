# XP Engine

XP Categories:
- **DXP**: Discipline XP
- **PXP**: Productivity XP
- **IXP**: Identity XP
- **AXP**: Alignment XP
- **MXP**: Mastery XP

## Belt Ranks
White → Yellow → Orange → Green → Blue → Purple → Brown → Black → Transcendent

Each rank has:
- XP thresholds
- Behavioral milestones
- Personalized identity notes
- Philosophy integration

## XP Flow

Action → Memory Item → XP Engine → Ledger → Insights → Dashboard

XP is earned across:
- Tasks
- Habits
- Journaling
- Coaching
- Conversations
- Achievements
- Planning
- Relationship interactions

XP must always feel:
- Earned
- Fair
- Motivating
- Clear
