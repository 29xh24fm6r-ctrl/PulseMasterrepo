# Philosophy Dojo

The user trains in life philosophies as if training martial arts.

## Schools

1. **Stoic Path**
2. **Samurai / Bushido**
3. **Taoist Flow**
4. **Warrior Spirit**
5. **Strategist Way**

## Belt Progression

Each school has its own:
- Belts
- Training exercises
- Micro-lessons
- Journaling prompts
- Scenario drills
- Roleplays
- Reflections

The philosophy engine ties directly into:
- Identity Engine
- XP Engine
- Coach feedback
- Weekly planning
