# Third Brain / Memory Continuum Engine

The Memory Engine captures all meaningful user activity:

## Memory Types

- Journals
- Conversations
- Tasks completed
- Calls
- Emails
- Reflections
- Milestones
- Daily logs
- Relationship moments

## Compression System

Daily → Weekly → Monthly → Quarterly → Yearly summaries.

Each compression contains:
- Key actions
- Emotional trends
- Achievements
- Challenges
- Patterns
- Predictions

## Retrieval

Memory is searchable by:
- Semantic meaning
- Time range
- Topic clusters
- People involved

## Proactive Surfacing

The system shows memories when:
- Relevant to current behavior
- Emotional patterns repeat
- Decisions mirror past trends
- User is at risk of repeating mistakes
